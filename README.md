# Internet of Things – Exercise Solutions

This repository contains my solutions for all course exercises.  
All Basic, Advanced, and Communication tasks are stored in one single repository, as required.

---

## 📁 Repository Structure

### 1_Basics/
Solutions to the Basic exercises:
- `1_1_HelloWorld.md`
- `1_2_Diode.md`
- `1_3_DiodeFade.md`
- `1_4_Button.md`

### 2_Advance/
Solutions to the Advanced exercises:
- `2_1_Analog.md` — from **5 Analog.pdf**
- `2_2_MoreTechniques.md` — from **6 more techniques.pdf**
- `2_3_AirCooler.md` — from **7 air cooler.pdf**
- `2_4_TransmitData_OPTIONAL.md` — from **8 transmit data OPTIONAL.pdf**

### 3_Communication/
Solutions to the Communication exercises:
- `3_1_CommunicationBasics.md` — from **9 communication basics.pdf**
- `3_2_AddingDatabase.md` — from **10 adding a database.pdf**

---

## 📌 Naming Rules

- Folder names match the exercise groups:
  - `1_Basics`
  - `2_Advance`
  - `3_Communication`

- File names follow the pattern:
  - `<groupNumber>_<taskNumber>_<optionalName>.md`
  - Example: `3_2_AddingDatabase.md`

This structure ensures the teacher can easily find each solution based on the exercise group and task number.

---

## 📊 Evaluation Criteria

- Basic tasks mostly done → **10 points**  
- Basic + Advanced tasks mostly done → **30 points**  
- Basic + Advanced + Communication tasks done → **50 points**

---

## ✔ Optional Task

Task 8 (Transmit Data) is optional.
