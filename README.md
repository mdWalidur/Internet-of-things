## 🏆 Final Project

### Smart Room Monitor 🌡️

**Real-time temperature and humidity monitoring system**

```
┌─────────────────────────────────────────────────────┐
│  ESP32 (Wokwi) ──→ MQTT ──→ Python Flask Server    │
│       ↑                            ↓                 │
│   DHT22 Sensor              Web Dashboard           │
│                            (Live Graph + Chart.js)  │
└─────────────────────────────────────────────────────┘
```

#### 🛠️ Tech Stack
- **Hardware:** ESP32 (Wokwi Simulator)
- **Sensor:** DHT22 (Temperature & Humidity)
- **Protocol:** MQTT (test.mosquitto.org)
- **Backend:** Python Flask + Socket.IO
- **Frontend:** Chart.js, HTML/CSS

#### ⚡ Quick Start

```bash
# 1️⃣ Open the hardware project
→ https://wokwi.com/projects/460027800656399361

# 2️⃣ Install dependencies
pip install -r requirements.txt

# 3️⃣ Start the server
python app.py

# 4️⃣ View dashboard
→ http://localhost:5000
```

#### 📁 Project Structure
```
Final Project/
├── sketch.ino              # ESP32 code
├── app.py                  # Flask server
├── dashboard.html          # Web interface
├── requirements.txt        # Python dependencies
└── templates/              # Flask templates
```
